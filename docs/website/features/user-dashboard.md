## User Dashboard

The User Dashboard gives listing owners a frontend interface to manage their listings, favorites, and reviews.

### Setup

The Setup Wizard creates a Dashboard page automatically. Or add the **User Dashboard** block to any page.

### Dashboard Sections

#### My Listings

- View all submitted listings with status badges (Published, Pending, Draft, Expired)
- Edit listings inline
- View listing statistics
- Delete listings

#### Favorites

- View saved/favorited listings
- Remove favorites
- Quick links to listing detail pages

#### Reviews

- View reviews received on your listings
- Reply to reviews
- Track average ratings

### Listing Management

From the dashboard, listing owners can:

1. **Edit** their listings (fields, images, description)
2. **Renew** expired listings
3. **Delete** listings they no longer want
4. **View analytics** on views and clicks (Pro)

### Access Control

- The dashboard requires user login
- Users only see their own listings
- Admin capabilities are not required
- The `edit_listora_listings` capability controls what users can do
