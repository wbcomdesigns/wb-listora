## Frontend Listing Submission

WB Listora allows users to submit listings from the frontend without accessing the WordPress admin.

### Setup

1. The Setup Wizard creates an "Add Listing" page automatically
2. Or add the **Listing Submission** block to any page
3. Configure submission settings under **Listora > Settings > Submissions**

### Submission Flow

The submission form is a multi-step process:

1. **Choose Type:** Select the listing type (Restaurant, Hotel, etc.)
2. **Basic Info:** Title, description, and featured image
3. **Type Fields:** Fields specific to the chosen type (address, phone, hours, etc.)
4. **Categories:** Select relevant categories and tags
5. **Preview & Submit:** Review before submitting

### Moderation

Configure how submitted listings are handled:

- **Auto-publish:** Listings go live immediately
- **Manual review:** Listings are held as "Pending" for admin approval
- **Edit approval:** Changes to existing listings require re-approval

### User Dashboard

After submission, users manage their listings from the Dashboard page:

- View all submitted listings with status
- Edit existing listings
- View listing analytics (Pro)
- Track review responses

### Settings

Under **Listora > Settings > Submissions:**

- **Require login:** Only registered users can submit
- **Moderation mode:** Auto-publish or manual review
- **Allowed types:** Which listing types accept submissions
- **Image limits:** Maximum number of gallery images
- **Expiration:** Days until a listing expires (0 = never)
