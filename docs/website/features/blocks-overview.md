## Blocks Overview

WB Listora provides 11 WordPress blocks built with the Interactivity API for fast, reactive interfaces without page reloads.

### Listing Search

The search block provides a command-palette-style search bar with:

- Keyword search with autocomplete suggestions
- Location search with "Near Me" geolocation
- Type filter tabs (All, Restaurant, Hotel, etc.)
- Advanced filters panel (category, features, price range, rating)
- Active filter pills showing applied filters

**Layouts:** Horizontal (default) or Stacked

### Listing Grid

The grid block displays listing cards in a responsive layout:

- **Grid view:** 1-4 columns with responsive breakpoints
- **List view:** Single column horizontal cards
- **Toolbar:** Result count, view toggle, sort dropdown
- **Pagination:** Page numbers with prev/next
- **Loading:** Skeleton placeholders during search
- **Empty state:** Friendly message with clear filters action

### Listing Detail

Automatically used on single listing pages. Shows:

- Hero gallery with thumbnails
- Tabbed content (Details, Reviews, Map, Contact)
- Sidebar with contact info, business hours, social links
- Share, Favorite, Claim, and Compare buttons
- Related listings section
- Schema.org JSON-LD markup

### Listing Reviews

Review system with:

- Rating summary (average, distribution chart)
- Review list with avatars, ratings, dates
- Review submission form with star rating
- Helpful votes and reporting
- Owner reply capability
- Multi-criteria ratings (Pro: food, service, ambiance, value)

### Other Blocks

- **Listing Map:** Interactive Leaflet map with clustering
- **Listing Submission:** Multi-step frontend form
- **User Dashboard:** Manage listings, favorites, reviews
- **Listing Categories:** Icon grid with counts
- **Listing Featured:** Featured listings display
- **Listing Calendar:** Event calendar view
